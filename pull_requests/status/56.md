### 5. Lint Configuration Verification

The `biome.jsonc` correctly extends `ultracite/biome/core`,
`ultracite/biome/next`, and `ultracite/biome/react`. The overrides in place are
intentional:

- `noExplicitAny: off` — allows `any` types where needed in this codebase
- `noUnknownAtRules: off` — required for Tailwind CSS custom at-rules
- `noConsole: off` — server-side logging is intentional
- `noMagicNumbers: off` — HTTP status codes are self-explanatory in context
- `useSortedClasses: off` — Tailwind class ordering is managed manually
- `noUnnecessaryConditions: off` — avoids false positives in conditional logic
- `useConsistentTypeDefinitions: off` — allows both `type` and `interface`

All 187 files pass lint and format with zero errors.
